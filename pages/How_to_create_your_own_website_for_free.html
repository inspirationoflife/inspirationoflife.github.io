Все веб-сайты делятся на статические и динамические. Мы будем создавать статический веб-сайт.
Для размещения веб-сайта в Интернете нужны хостинг и домен. Мы получим их бесплатно с помощью GitHub Pages.
Статический веб-сайт состоит из отдельных веб-страниц, которые пишутся на языках HTML и CSS. Для их изучения я рекомендую следующие пособия:

Шаг 1. Регистрируемся на GitHub

Как уже было сказано, размещать наш сайт в Интернете мы будем с помощью GitHub Pages. Поэтому сначала нам нужно зарегистрироваться на GitHub. Для этого вы можете перейти на главную страницу и нажать Sign up в левом верхнем углу, либо воспользоваться прямой ссылкой.
