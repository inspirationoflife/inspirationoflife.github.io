Как создать свой сайт бесплатно

Сегодня мы будем бесплатно создавать свой собственный веб-сайт. Для этого нам даже не нужен ноутбук или ПК: подойдёт обычный сенсорный телефон (смартфон). Собственно, я уже несколько лет так и работаю, поэтому и инструкцию я тоже буду показывать для телефона. Впрочем, процесс создания сайта на телефоне (данным методом) по своей сути ничем не отличается от создания сайта на ПК или ноутбуке. Только лишь расположение кнопок может быть немного разным. Для этого нам даже не обязательно иметь ноутбук или ПК: лично мне уже несколько лет спокойно хватает телефона. Для этого нам предстоит:
  1. Выучить HTML
  2. Выучить CSS
  3. Научиться пользоваться ГитХабом
HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — это каркас, на котором будут строиться все наши веб-страницы.

CSS (англ. Cascading Style Sheets «каскадные таблицы стилей») — это дополнительный инструмент, который позволяет управлять внешним видом наших страниц.

ГитХаб — это сервис, через который мы будем создавать и редактировать свой веб-сайт. GitHub предоставит нам бесплатный хостинг (размещение нашей информации в Интернете) и бесплатный домен (URL-адрес).

Конечно, курс изучения HTML и CSS предусмотрен школьной программой 11-ого класса. Но во-первых, школьная программа даёт слишком недостаточные знания, а во-вторых вёрстку можно изучить гораздо раньше. Лично я для того, чтобы выполнить первые две задачи, недели на две ушёл с головой в чтение списков всех тегов HTML и свойств CSS (при этом было это примерно в классе 7-ом-8-ом). Всё остальное так: по мелочи. Это как в изучении английского (или любого другого иностранного) языка: главное — накопить словарный запас, а остальное уже на втором месте.
