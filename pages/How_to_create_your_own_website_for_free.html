Как создать свой сайт бесплатно

Введение

Веб-сайты делятся на два типа: динамические и статические. О разнице между ними можно прочитать в следующих статьях:
  • https://blog.ishosting.com/ru/static-vs-dynamic-websites?hs_amp=true
  • https://web-revenue.ru/sajtostroenie/static-and-dynamic-website
  • https://moolkin.ru/joomla/cms/staticheskie-i-dinamicheskie-web-sayty-v-chyom-raznitsa
  • https://www.figma.com/resource-library/static-vs-dynamic-website
В данной статье речь пойдёт о создании статического сайта. Для этого даже не обязательно иметь ноутбук или ПК. Все действия выполняются прямо в браузере, так что для разработки сайта будет достаточно и сенсорного телефона. При этом вам также не обязательно изучать языки программирования. Основной язык, который вам нужно будет выучить, — это HTML.
HTML (эйч-ти-эм-эл; от англ. HyperText Markup Language — «язык гипертекстовой разметки») — это язык, на котором пишутся веб-страницы. HTML является универсальной формой представления информации, сочетающей в себе все возможные элементы: текст, изображения, видео, таблицы и т. д.
Практически всегда вместе с HTML используется CSS.
CSS (си-эс-эс; от англ. Cascading Style Sheets — «каскадные таблицы стилей») — это язык, позволяющий настроить внешний вид содержимого страницы (контента).
Для изучения этих двух языков я рекомендую следующие сайты:
  • htmlbook: учебник HTML, справочник HTML, справочник CSS;
  • basicweb: учебник HTML, справочник HTML, справочник CSS;
  • mdn: учебник HTML, справочник HTML, справочник CSS;
  • schoolsw3: учебник HTML, справочник HTML, справочник CSS;
  • и другие.
