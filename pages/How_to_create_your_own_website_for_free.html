Как создать свой сайт бесплатно

Сегодня мы будем бесплатно создавать свой собственный веб-сайт. Для этого нам предстоит:
  1. Выучить HTML
  2. Выучить CSS
  3. Научиться пользоваться ГитХабом
Конечно, курс изучения HTML и CSS предусмотрен школьной программой 11-ого класса. Но во-первых, школьная программа даёт слишком недостаточные знания, а во-вторых вёрстку можно изучить гораздо раньше. Лично я для того, чтобы выполнить первые две задачи, недели на две ушёл с головой в чтение списков всех тегов HTML и свойств CSS (при этом было это примерно в классе 7-ом-8-ом). Всё остальное так: по мелочи. Это как в изучении английского (или любого другого иностранного) языка: главное — накопить словарный запас, а остальное уже на втором месте.
