Как создать свой сайт бесплатно

Что представляет собой данная инструкция?

Данная инструкция представляет собой несколько видоизменённое официальное руководство GitHub Pages. GitHub Pages — это функция GitHub, которая предоставляет бесплатный хостинг и домен для размещения статического веб-сайта в Интернете. Статический веб-сайт состоит из отдельных веб-страниц, так что имейте в виду, что для того, чтобы создать сайт по данной инструкции, вам необходимо обладать знаниями HTML и CSS. Для изучения этих двух языков я рекомендую вам следующие пособия:
  • htmlbook: учебник HTML, справочник HTML, справочник CSS
  • basicweb: учебник HTML, справочник HTML, справочник CSS
  • mdn: учебник HTML, справочник HTML, справочник CSS
  • schoolsw3: учебник HTML, справочник HTML, справочник CSS

Шаг 1. Регистрируемся на GitHub

Сначала нам надо зарегистрироваться на GitHub. Для этого вы можете перейти на главную страницу и нажать Sign up for GitHub либо воспользоваться прямой ссылкой на страницу регистрации. Чтобы создать аккаунт, нужен только адрес электронной почты.

Шаг 2. Создаём репозиторий

После создания учётной записи вы вернётесь на главную страницу, но теперь вы будете видеть перед собой панель инструментов (Dashboard). Теперь нам нужно создать репозиторий для нашего сайта. Для этого вы можете нажать кнопку Create repository в разделе «Repositories» либо воспользоваться прямой ссылкой на страницу создания репозитория.
Для того, чтобы активировать функцию GitHub Pages, в поле «Repository name» вы должны ввести «nickname.github.io», где «nickname» — имя пользователя, которое вы выбрали на предыдущем шаге, указанное в нижнем регистре. Например, если при создании аккаунта в разделе «Username» вы указали «RickOwens», то тогда в поле «Repository name» вы должны ввести «rickowens.github.io».
