Как создать свой сайт бесплатно

В этой статье вы узнаете, как обзавестись своим собственным веб-сайтом совершенно бесплатно. Для этого даже не обязательно иметь ноутбук или ПК! Все действия выполняются прямо в браузере, так что для разработки сайта будет достаточно и сенсорного телефона.

Этот метод подойдет тем, кому просто нужно иметь место для хранения информации в открытом доступе и кто не хочет заморачиваться со сложностями программирования.

Введение. Комбинация «HTML и CSS + GitHub»

HTML (эйч-ти-эм-эл; от англ. HyperText Markup Language — «язык гипертекстовой разметки») — это язык, на котором пишутся веб-страницы. HTML является универсальной формой представления информации, сочетающей в себе все возможные элементы: текст, изображения, видео, таблицы и т. д.
Практически всегда вместе с HTML используется CSS.
CSS (си-эс-эс; от англ. Cascading Style Sheets — «каскадные таблицы стилей») — это язык, позволяющий настроить внешний вид содержимого страницы (контента).
Для изучения этих двух языков я рекомендую следующие сайты:
  • htmlbook: учебник HTML, справочник HTML, справочник CSS;
  • basicweb: учебник HTML, справочник HTML, справочник CSS;
  • mdn: учебник HTML, справочник HTML, справочник CSS;
  • schoolsw3: учебник HTML, справочник HTML, справочник CSS;
  • и другие.
Для размещения нашего сайта в Интернете нам понадобится хостинг и домен. И то, и другое можно получить бесплатно с помощью GitHub. Для этого мы воспользуемся функцией GitHub Pages.

Шаг 1. Регистрируемся на GitHub

GitHub Pages — это функция GitHub, позволяющая пользователю создать сайт из репозитория.

Если вы уже знакомы с этими двумя языками, но не знаете, как выложить свой сайт в Интернет, то можете переходить дальше. Если же вы впервые сталкиваетесь с HTML и CSS, то вы можете пойти двумя путями: сначала изучить HTML и CSS, а потом перейти к созданию сайта, либо же сначала создать сайт, а потом приступить к изучению HTML и CSS. Неплохо, если на данный момент вы уже владеете этими языками. Но даже если нет, это не беда, так как вы можете сначала посмотреть, как создаётся сайт на GitHub, а потом приступить к изучению HTML и CSS.
