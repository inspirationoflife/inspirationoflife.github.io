Как создать свой сайт бесплатно

HTML (эйч-ти-эм-эл; от англ. HyperText Markup Language — «язык гипертекстовой разметки») — это язык, на котором пишутся веб-страницы. HTML является универсальной формой представления информации, сочетающей в себе все возможные элементы: текст, изображения, видео, таблицы и т. д.

CSS (си-эс-эс; от англ. Cascading Style Sheets — «каскадные таблицы стилей») — это дополнение к HTML

GitHub — это сервис, через который мы будем создавать и редактировать свой веб-сайт. GitHub предоставит нам бесплатный хостинг и бесплатный домен.

Что же касается третьей задачи, то ГитХабом вы научитесь пользоваться по ходу статьи.

Этих трёх навыков вполне хватит для создания простенького веб-сайта. Итак, начнём!

GitHub предоставляет всем желающим бесплатный хостинг и бесплатный домен через функцию GitHub Pages.

Интерфейс GitHub периодически меняется, так что скриншоты со временем могут в той или иной мере устаревать. Однако в основном это касается лишь мелких делалей; в целом весь механизм остаётся тот же.

Данный сервис имеет такую функцию, которая называется GitHub Pages. С помощью неё-то мы и будем создавать наш сайт

Создание статического веб-сайта предполагает наличие знаний HTML и CSS.
