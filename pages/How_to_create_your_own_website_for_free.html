Как создать свой сайт бесплатно

В этой статье вы узнаете, как сделать собственный веб-сайт с помощью GitHub Pages (GitHub Docs, Wikipedia). GitHub Pages — это функция GitHub, которая предоставляет бесплатный хостинг и домен для размещения веб-сайта в Интернете.

Быстрый старт

В этой пошаговой инструкции мы создаём главную страницу сайта. Она будет представлять собой файл index.html, содержащий фразу «Hello, World!».

Шаг 1. Регистрируемся на GitHub

Сначала нам надо зарегистрироваться на GitHub. Для этого вы можете перейти на главную страницу и нажать Sign up for GitHub либо воспользоваться прямой ссылкой на страницу регистрации. Чтобы создать аккаунт, нужен только адрес электронной почты.

Шаг 2. Создаём репозиторий

После создания учётной записи вы вернётесь на главную страницу и увидите перед собой панель инструментов (Dashboard). Теперь нам нужно создать репозиторий для нашего сайта. Для этого вы можете нажать кнопку Create repository в разделе «Repositories» либо воспользоваться прямой ссылкой на страницу создания репозитория.

Для того, чтобы активировать функцию GitHub Pages, в поле «Repository name» вы должны ввести «nickname.github.io», где «nickname» — выбранное вами на предыдущем шаге имя пользователя, указанное в нижнем регистре. Например, если при создании аккаунта в разделе «Username» вы указали «RickOwens», то тогда в поле «Repository name» вы должны ввести «rickowens.github.io».

Все остальные поля вы можете оставить без изменений и нажать Create repository.

Шаг 3. Добавляем лицензию

После нажатия кнопки Create repository, вы попадёте на страницу созданного репозитория. Теперь нам нужно добавить лицензию.

На мой взгляд, самый лучший вариант — это WTFPL license. Вы можете сравнить WTFPL с другими лицензиями по этой таблице, а также найти более подробную информацию о ней здесь:

  • Официальный сайт
  • Википедия
  • choosealicense

Для того, чтобы добавить лицензию в ваш репозиторий, найдите текст «Get started by creating a new file or uploading an existing file. We recommend every repository include a README, LICENSE, and .gitignore.» и нажмите на слово «LICENSE». Вы окажетесь на странице создания файла лицензии. Откройте текст лицензии WTFPL, скопируйте его и вставьте в поле «Enter file contents here». После этого нажмите Commit changes... и затем ещё раз Commit changes.

Шаг 4. Создаём главную страницу сайта

После создания первого файла страница вашего репозитория примет свой стандартный вид. Вы увидите файл лицензии в списке всех файлов репозитория (он называется «LICENSE.md»). Теперь нам нужно создать главную страницу сайта. Для этого выполните следующие действия:

  1. Нажмите на кнопку , которая находится на одном уровне с кнопкой main
  2. Нажмите Create new file
  3. В поле «Enter file contents here» введите любой текст, который вы хотите отобразить на главной странице сайта. По идее, он должен представлять собой html-код, но для первого раза вы можете просто вставить сюда стандартную фразу «Hello, World!» и посмотреть, как она будет выглядеть на главной странице сайта.
  4. В поле «Name your file...» введите «index.html»
  5. Нажмите Commit changes... и затем Commit changes

После этого главная страница вашего сайта будет доступна по ссылке «https://rickowens.github.io».

Шаг 5. Добавляем ссылку на сайт на страницу репозитория

Осталось сделать последний штрих — добавить ссылку на наш сайт на страницу репозитория. Для этого выполните следующие действия:

  1. Перезагрузите страницу, на которой вы оказались после создания файла «index.html»
  2. Внизу с правой стороны найдите кнопку . Она находится над словом «Releases».
  3. Поставьте галочку напротив пункта «Use your GitHub Pages website»
  4. Нажмите Save changes
