Создавать мы будем так называемый статический веб-сайт. Процесс его создания имеет две основные составляющие: написание отдельных веб-страниц на HTML и CSS и размещение (хостинг) сайта в Интернете. В остальном данная процедура ничем не отличается от редактирования папки с файлами.
Что касается первой составляющей, то о ней достаточно много информации в Интернете. Вот несколько примеров сайтов, с помощью которых вы можете изучить HTML и CSS: Поэтому в данной статье в основном будет рассматриваться вторая составляющая.

Шаг 1. Регистрируемся на GitHub

Как уже было сказано, размещать наш сайт в Интернете мы будем с помощью GitHub Pages. Поэтому сначала нам нужно зарегистрироваться на GitHub. Для этого вы можете перейти на главную страницу и нажать Sign up в левом верхнем углу, либо воспользоваться прямой ссылкой.
