Как создать свой сайт бесплатно

Сегодня вы научитесь, как бесплатно создать свой собственный веб-сайт. Для этого вам даже не нужен ноутбук или ПК! Хватит и сенсорного телефона.

Создавать мы будем так называемый статический веб-сайт, поэтому вам предстоит:

  1. Выучить HTML
  2. Выучить CSS
  3. Научиться пользоваться ГитХабом

HTML (эйч-ти-эм-эл; от англ. HyperText Markup Language — «язык гипертекстовой разметки») — это каркас, на котором будут строиться все наши веб-страницы.

CSS (си-эс-эс; от англ. Cascading Style Sheets — «каскадные таблицы стилей») — это дополнительный инструмент, который позволяет настроить внешний вид наших страниц.

GitHub — это сервис, через который мы будем создавать и редактировать свой веб-сайт. GitHub предоставит нам бесплатный хостинг и бесплатный домен.

Но во-первых, школьная программа даёт слишком недостаточные знания, а во-вторых вёрстку можно изучить гораздо раньше. Лично я для того, чтобы изучить эти языки, недели на две ушёл с головой в чтение списков всех тегов HTML и свойств CSS (при этом было это примерно в классе 7-ом-8-ом). Всё остальное так: по мелочи. GitHub предоставляет всем желающим бесплатный хостинг и бесплатный домен через функцию GitHub Pages.

Интерфейс GitHub периодически меняется, так что скриншоты со временем могут в той или иной мере устаревать. Однако в основном это касается лишь мелких делалей; в целом весь механизм остаётся тот же.

Данный сервис имеет такую функцию, которая называется GitHub Pages. С помощью неё-то мы и будем создавать наш сайт

Если же нет, то в этом нет ничего страшного: вы можете сначала выполнить все шаги нижеследующей инструкции по созданию вашего первого сайта и вывести на экран браузера ваше первое «Hello, World!», убедившись в том, что процесс создания сайта на ГитХабе довольно лёгкий, а потом уже приступить к изучению HTML и CSS.
