Немного теории

Виды сайтов

Все веб-сайты делятся на два типа: динамические и статические. Динамические сайты создаются с помощью конструкторов, которые называются CMS. Например:
  • uCoz
  • WordPress
  • Google Sites
Однако динамические сайты мы пока можем отложить в сторону, потому что они предназначены в основном для крупных и сложных проектов, предполагающих активное взаимодействие пользователей и сервера, таких как интернет-магазины.

HTML и CSS

Если вы хотите создать статический веб-сайт, вам предстоит познакомиться с такими понятиями, как HTML и CSS.
HTML (эйч-ти-эм-эл; от англ. HyperText Markup Language — «язык гипертекстовой разметки») — это язык, на котором пишутся веб-страницы. HTML является универсальной формой представления информации, сочетающей в себе все возможные элементы: текст, изображения, видео, таблицы и т. д.
CSS (си-эс-эс; от англ. Cascading Style Sheets — «каскадные таблицы стилей») — это дополнение к HTML, позволяющее настраивать внешний вид содержимого вашей страницы (т. е. контента).
В качестве учебников этих двух языков я могу предложить вам следующие сайты:
  • htmlbook: HTML, CSS
  • basicweb: HTML, CSS
  • mdn: HTML, CSS
  • schoolsw3: HTML, CSS

GitHub

Однако, знать HTML и CSS для создания веб-сайта недостаточно. Для того, чтобы разместить наш сайт в Интернете, нам нужен хостинг и домен. И то и другое можно получить бесплатно. Для этого мы воспользуемся таким сайтом, как GitHub.
GitHub — это крупнейший в мире веб-сервис для хостинга IT-проектов и их совместной разработки.
А так как вёрстка веб-страниц тоже входит в IT, то GitHub предоставляет для этого специальный инструмент — GitHub Pages.

GitHub Pages

GitHub Pages — это функция GitHub, позволяющая создать по одному сайту на одного пользователя. В этой статье речь пойдёт как раз о создании сайта на GitHub Pages.
