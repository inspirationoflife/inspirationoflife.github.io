Как создать свой сайт бесплатно

В этой статье вы узнаете, как обзавестись своим собственным веб-сайтом совершенно бесплатно. Для этого даже не обязательно иметь ноутбук или ПК! Все действия выполняются прямо в браузере, так что для разработки сайта будет достаточно и сенсорного телефона.

Введение. Комбинация «HTML и CSS + GitHub»

HTML (эйч-ти-эм-эл; от англ. HyperText Markup Language — «язык гипертекстовой разметки») — это язык, на котором пишутся веб-страницы. HTML является универсальной формой представления информации, сочетающей в себе все возможные элементы: текст, изображения, видео, таблицы и т. д.
Практически всегда вместе с HTML используется CSS.
CSS (си-эс-эс; от англ. Cascading Style Sheets — «каскадные таблицы стилей») — это язык, позволяющий настроить внешний вид содержимого страницы (контента).
Для изучения этих двух языков я рекомендую следующие сайты:
  • htmlbook: учебник HTML, справочник HTML, справочник CSS;
  • basicweb: учебник HTML, справочник HTML, справочник CSS;
  • mdn: учебник HTML, справочник HTML, справочник CSS;
  • schoolsw3: учебник HTML, справочник HTML, справочник CSS;
  • и другие.
GitHub Pages — это бесплатный способ разместить сайт в Интернете.

Шаг 1. Регистрируемся на GitHub

Однако, знать HTML и CSS для создания веб-сайта недостаточно. Чтобы другие люди могли получить доступ к нашему сайту, нам необходимо разместить его в Интернете. В этом деле нам может помочь такой инструмент, как GitHub Pages. GitHub Pages — это функция GitHub, позволяющая пользователю создать сайт из репозитория. Итак, перейдём к практической инструкции. Чтобы создать статический веб-сайт, вам необходимо ознакомиться с такими понятиями, как HTML и CSS.
Если вы уже знакомы с этими двумя языками, но не знаете, как выложить свой сайт в Интернет, то можете переходить дальше. Если же вы впервые сталкиваетесь с HTML и CSS, то вы можете пойти двумя путями: сначала изучить HTML и CSS, а потом перейти к созданию сайта, либо же сначала создать сайт, а потом приступить к изучению HTML и CSS. Неплохо, если на данный момент вы уже владеете этими языками. Но даже если нет, это не беда, так как вы можете сначала посмотреть, как создаётся сайт на GitHub, а потом приступить к изучению HTML и CSS. Однако, знать HTML и CSS для создания веб-сайта недостаточно. Для того, чтобы разместить наш сайт в Интернете, нам нужен хостинг и домен. И то и другое можно получить бесплатно. Для этого мы воспользуемся таким сайтом, как GitHub. GitHub позволяет создавать свой сайт прямо в браузере. Для этого мы воспользуемся GitHub Pages. В этой статье речь пойдёт как раз о создании сайта на GitHub Pages.
