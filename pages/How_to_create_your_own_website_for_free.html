Как создать свой сайт бесплатно

Немного теории

Виды сайтов

Все веб-сайты делятся на два типа: динамические и статические. Динамические сайты создаются с помощью конструкторов, которые называются CMS. Например:
  • uCoz
  • WordPress
  • Google Sites

HTML и CSS

Для того чтобы создать статический веб-сайт вам предстоит познакомиться с такими понятиями, как HTML и CSS.
HTML (эйч-ти-эм-эл; от англ. HyperText Markup Language — «язык гипертекстовой разметки») — это язык, на котором пишутся веб-страницы. HTML является универсальной формой представления информации, сочетающей в себе все возможные элементы: текст, изображения, видео, таблицы и т. д.
CSS (си-эс-эс; от англ. Cascading Style Sheets — «каскадные таблицы стилей») — это язык, позволяющий настроить внешний вид содержимого страницы (контента).
Процесс создания сайта с помощью HTML и CSS называется вёрстка (англ. front-end web development — «разработка веб-приложений на стороне клиента»).
Если вы уже знакомы с этими двумя языками, но не знаете, как выложить свой сайт в Интернет, то можете переходить дальше. Если же вы впервые сталкиваетесь с HTML и CSS, то вы можете пойти двумя путями: сначала изучить HTML и CSS, а потом перейти к созданию сайта, либо же сначала создать сайт, а потом приступить к изучению HTML и CSS.
  • htmlbook: HTML, CSS
  • basicweb: HTML, CSS, селекторы
  • mdn: HTML, CSS
  • schoolsw3: HTML, CSS
Неплохо, если на данный момент вы уже владеете этими языками. Но даже если нет, это не беда, так как вы можете сначала посмотреть, как создаётся сайт на GitHub, а потом приступить к изучению HTML и CSS.

GitHub Pages

Однако, знать HTML и CSS для создания веб-сайта недостаточно. Для того, чтобы разместить наш сайт в Интернете, нам нужен хостинг и домен. И то и другое можно получить бесплатно. Для этого мы воспользуемся таким сайтом, как GitHub.
GitHub — это крупнейший в мире веб-сервис для хостинга IT-проектов и их совместной разработки.
GitHub позволяет создавать свой сайт прямо в браузере. При этом нет необходимости в ноутбуке или ПК — будет достаточно даже сенсорного телефона. Для этого мы воспользуемся GitHub Pages.
GitHub Pages — это функция GitHub, позволяющая создать по одному сайту на одного пользователя.
В этой статье речь пойдёт как раз о создании сайта на GitHub Pages.
